# TJK At Yarışı Analiz Paneli

سایت زنده‌ای که هر وقت باز بشه (یا وقتی کش ۱۰ دقیقه‌ای منقضی بشه) از **tjk.org**
اطلاعات مسابقات امروز/فردا، اسب‌ها، جوکی‌ها، و ۶ نتیجه آخر هر اسب رو می‌گیره،
تو حافظه (in-memory) نگه می‌داره، و یه درصد شانس برد تخمینی نشون می‌ده.

## راه‌اندازی

```bash
npm install
npm start
```
بعد مرورگر رو باز کن: `http://localhost:3000`

## ساختار پروژه

```
server.js         # بک‌اند Express: fetch + parse + cache + API + محاسبه شانس
public/index.html # صفحه اصلی
public/app.js      # فرانت‌اند: fetch از API خودمون و رندر جدول
public/style.css   # استایل
```

## API Endpoints

- `GET /api/races?day=today|tomorrow` — لیست کامل شهرها/کوشوها/اسب‌ها + شانس برد
- `GET /api/horse/:atId` — تاریخچه ۶ مسابقه آخر یه اسب خاص
- `POST /api/cache/clear` — پاک کردن کش و اجبار به گرفتن داده تازه

## مدل تخمین شانس برد (فعلاً ساده)

```
score = 0.7 × HP + 0.3 × (1 / میانگین رتبه ۶ مسابقه آخر) × 100
probability = score اسب / مجموع score همه اسب‌های اون مسابقه
```

این یه نقطه شروعه، نه یه مدل نهایی. پیشنهاد بعدی: اضافه کردن نرخ برد جوکی،
تناسب با مسافت/پیست خاص، و backtesting روی نتایج قبلی برای تنظیم ضرایب.
